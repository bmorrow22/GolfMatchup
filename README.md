# GolfMatchup ⛳️
A React Native app for live tournament tracking at Torrey Pines.

### Current Features:
- **Auto-Balance**: Snake draft logic to pair 16 players into balanced teams.
- **Live Scorecard**: Real-time entry with handicap "pops" (strokes) calculated per hole.
- **Skins Logic**: Automated net-skin tracking with tie detection.

### Tech Stack:
- Expo / React Native
- TypeScript
- Expo Router (File-based routing)
